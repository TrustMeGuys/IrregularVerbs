Первое задание:
	На экран будет выведено неправильный глагол в первой форме.
	Необходимо будет ввести 2 оставшихся формы глагола.
	У глагола "be" несколько форм - их необходимо ввести через "/".
Второе задание:
	На экран будет выведено предложение необходимо будет его дополнить.
	Также будет выведено определенное количество вариантов,которое надо выбрать.
	Вводить необходимо только цифру ответа.
Третье задание:
	На экран будет выведено предложение, а также различные варианты перевода.
	Вам необходимо перевести, а затем выбрать и ввести в правильном порядке номера слов перевода.
	Вариантов перевода может быть несколько.
